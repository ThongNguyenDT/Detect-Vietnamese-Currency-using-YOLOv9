<h1 align="center">Detect Vietnamese Currency using YOLOv9 with DeepSORT</h1>

<h2 align="center">Overview</h2>
The project "Detect Vietnamese Currency using YOLOv9 with DeepSORT to Calculate Total Money in Camera Screen" aims to develop an automated system that can accurately detect and count Vietnamese currency notes in real-time using a camera. This system integrates advanced object detection and tracking algorithms to identify various denominations of Vietnamese currency and compute the total amount of money visible in the camera's field of view.
